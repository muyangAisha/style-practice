网格布局（Grid）是最强大的 CSS 布局方案

与flex的区别

  • Flex 布局是轴线布局，只能指定"项目"针对轴线的位置，可以看作是一维布局。
  • Grid 布局则是将容器划分成"行"和"列"，产生单元格，然后指定"项目所在"的单元格，可以看作是二维布局。Grid 布局远比 Flex 布局强大。

基本概念

  • 1. 最外层的div元素称为"容器"（container）,内层的顶级div子元素称为"项目"（item）。
  • 2. 容器里面的水平区域称为"行"（row），垂直区域称为"列"（column）。
  • 3. 行和列的交叉区域，称为"单元格"（cell）。正常情况下，3行3列会产生9个单元格（n * m => 3*3=9）
  • 4. 划分网格的线，称为"网格线"（grid line）。正常情况下，4 * 4 的网格，共有5根水平网格线和5根垂直网格线。

grid-template-columns 和 grid-template-rows

  • grid-template-columns 属性定义每一列的列宽，可以是绝对单位(100px)，也可以是百分比(33.33%)
  • grid-template-rows 属性定义每一行的行高，可以是绝对单位(100px)，也可以是百分比(33.33%)
  • repeat()函数，网格很多,复写同样的值非常麻烦 ===> repeat(3, 33.33%)，第一个参数是重复的次数（上例是3），后面参数是所要重复的值

  • example3

  • repeat(2, 100px 20px 80px) ===> 第一列和第四列的宽度为100px，第二列和第五列为20px，第三列和第六列为80px.
